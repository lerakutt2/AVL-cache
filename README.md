# diplom
## Как запустить программу?
*Примечание.* Способы не тестировались, но работают локально (у меня на компьютере), поэтому высокая вероятность возникновения неполадок.

**Для запуска программы необходимо положить [папку c файлом ggParameters.txt](https://disk.jinr.ru/index.php/s/QAx8i5BmMXJRdkY) в основную папку программы (/diplom)**
### 1 способ. Linux
1. Открыть терминал Linux (или WSL)
2. Перейти в директорию проекта
3. Ввести следующие команды

```
mkdir -p build
cd build
cmake ..
make
./diplom_new
```
5. Далее можно запускать командами
```
make
./diplom_new
```
### 2 способ. Visual Studio
1. Открыть папку проекта с помощью Visual Studio
2. Собрать все (F6) и запустить (F5)

## Что делает программа на данный момент:
- Считывает параметры функции GG из файла ggParameters.txt
- Считает результаты функции для разных параметров с помощью дерева
- Считает результаты функции для разных параметров с помощью прямого вызова функции
- Сравнивает результаты: дерево быстрее даже при 10 вызовах функции (10 наборов параметров, 4 из них повторяются)
